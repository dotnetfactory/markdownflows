# MarkdownFlows Documentation

This folder contains documentation and assets for the MarkdownFlows project.

## Screenshots

Add screenshots to this folder to showcase the application:

- `screenshot.png` - Main application screenshot (referenced in README)
- `ai-generation.png` - AI diagram generation demo
- `export.png` - SVG export feature

## Taking Screenshots

To take good screenshots:

1. Use a clean diagram with meaningful content
2. Make sure both editor and preview panels are visible
3. Use a window size around 1400x900 for optimal aspect ratio
4. Consider using both light and dark themes
